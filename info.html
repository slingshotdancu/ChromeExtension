Chrome Diff Checker Extension

This file is documentation intended to provide additional information about the ISS Chrome Diff Checker Extension, and is intended for use by ISS Norman personnel.

The Chrome Diff Checker Extension is currently only compatible with Google Chrome.

This Extension is enabled for specific domains including sec.gov, otcmarkets.com, and google.com, and the page action and popup function are not currently enabled for other domains.

Installation

  1. Open Chrome and the Settings Menu(Alt+F) -> More Tools -> Extensions

  2. Enable Developer Mode by clicking the switch at the top-right of the page.

  3. Click Load Unpacked.

  4. Select Folder located at L:\Data_Ops\Comp\Diff Checker

  5. The ISS Logo will appear as a new icon in the menu bar. Close the settings window and the plugin is ready for use!

Use and Operation

  • To compare two similar paragraphs, pages must be located in two separate windows.

  • The popup compares input in real time, and is automatically displayed at the end of the page.

  • Start typing in the provided fields-common words will appear with no formatting, while unique words will have a border color corresponding to the color of their respective input region.

  • Select and Copy(Ctrl+C) the first paragraph.

  • Open the extension window, and Paste(Ctrl+V) into first field.

  • Navigate to the second window, and copy the second paragraph.

  • Paste into the second field, and differences will be colorized below.
